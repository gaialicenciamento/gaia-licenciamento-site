# Gaia Licenciamento Site

Site institucional estático da Gaia Licenciamento.

## Arquivos

- index.html
- style.css

## Publicar

No GitHub, acesse Settings, depois Pages. Em Source, selecione Deploy from a branch. Escolha branch main e pasta root. Salve.

Endereço esperado após publicar:

https://gaialicenciamento.github.io/gaia-licenciamento-site/

## Pendências

- Trocar o link de contato pelo WhatsApp real.
- Inserir a verificação da Meta quando o código for gerado.
- Configurar domínio próprio depois, se necessário.
